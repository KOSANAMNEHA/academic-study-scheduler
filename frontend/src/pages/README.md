# 📚 Academic Study Scheduler

A full-stack Smart Study Planner web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This application helps students plan their study schedule, track progress, manage materials, and stay motivated.

---

## 🚀 Features

### 🔐 Authentication
- User Registration
- Secure Login (JWT Authentication)
- Protected Routes
- Logout functionality

### 📅 Study Planner
- Add weekly study tasks
- Set priority (High, Medium, Low)
- Edit and delete tasks
- Weekly timetable view

### ⏱ Pomodoro Timer
- Focus sessions for effective studying
- Track number of Pomodoro sessions

### 📊 Progress Tracker
- Track completed tasks
- Track pending tasks
- Study hours statistics

### 📁 Study Materials
- Upload study materials (PDFs)
- Store subject-wise materials
- View PDFs directly in the application
- Delete materials

### 💡 Daily Motivation
- Random motivational quotes
- New quote on every login

### 📊 Dashboard
- Total study hours
- Completed tasks
- Pending tasks
- Pomodoro sessions
- Today's tasks overview

## ⚙ Installation

### 1️⃣ Clone the repository
https://github.com/yourusername/academic-study-scheduler.git
### 2️⃣ Install backend dependencies
cd backend
npm install
### 3️⃣ Install frontend dependencies


cd frontend
npm install

## ▶ Running the Application

### Start Backend


cd backend
node server.js


### Start Frontend


cd frontend
npm start


Application will run at:


http://localhost:3000
